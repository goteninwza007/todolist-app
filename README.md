# Todo App

A full-stack Todo application built with React + Go/Gin + PostgreSQL

## Tech Stack

### Client
- React + TypeScript
- Vite
- Tailwind CSS
- React Context API

### Server
- Go + Gin
- GORM
- PostgreSQL

## Prerequisites
- Docker + Docker Compose
- Node.js + pnpm (for local client dev)
- Go 1.23+ (for local server dev)

## Getting Started
```bash
docker compose up --build
```

- Client: http://localhost:3000
- Server: http://localhost:8080
